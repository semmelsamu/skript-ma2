Es sei $z = (a, b)$ eine komplexe Zahl. Dann ist
$$z = (a, 0) + (0, b) = (a, 0) + b(0, 1) \quad \text{\enquote{=}} \quad a + b \cdot i$$
die \term{Normaldarstellung} von $z$.
\begin{itemize}
    \item $a = \Re(z)$ heißt \term{Realteil} von $z$.
    \item $b = \Im(z)$ heißt \term{Imaginärteil} von $z$.
\end{itemize}
