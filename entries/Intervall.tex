In einem angeordneten kommutativen Körper $(K, +, \cdot)$ bezeichnet man die Mengen
$$[a, b] := \{x \in K \ | \ a \leq x \leq b \},$$
$$[a, b) := \{x \in K \ | \ a \leq x < b \},$$
$$(a, b] := \{x \in K \ | \ a < x \leq b \},$$
$$(a, b) := \{x \in K \ | \ a < x < b \}$$
für $a, b \in K$ als \term{Intervalle}.
\begin{itemize}
    \item $[a, b]$ heißt \term{abgeschlossen},
    \item $(a, b)$ heißt \term{offen},
    \item die beiden anderen heißen \term{halboffen}.
\end{itemize}
